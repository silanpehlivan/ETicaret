
🛒 ETicaret Projesi

Bu proje, Web Tabanlı Programlama dersi etkinlikleri kapsamında, ASP.NET Core MVC mimarisinin temellerini ve modern web geliştirme süreçlerini pratik etmek amacıyla geliştirilmiş başlangıç seviyesinde bir e-ticaret uygulamasıdır.

🛠️ Teknik Özellikler

Platform: .NET / ASP.NET Core

Dil: C#

Mimari: Model-View-Controller (MVC)

Geliştirme Ortamı: Visual Studio 2022

📁 Proje Yapısı

Proje, standart MVC klasör yapısı üzerine inşa edilmiştir:

Controllers: Uygulama mantığı ve sayfa yönlendirmelerinin yönetimi.

Models: Veri modelleri ve iş kurallarının tanımlanması.

Views: Kullanıcı arayüzü (UI) bileşenleri.

wwwroot: Statik dosyaların (CSS, JavaScript, resimler) yönetimi.

🎯 Kazanımlar

MVC (Model-View-Controller) tasarım deseninin uygulanması.

Backend ve Frontend bileşenleri arasındaki koordinasyonun sağlanması.

.json tabanlı yapılandırma (appsettings.json) dosyalarının kullanımı.
